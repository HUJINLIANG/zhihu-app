# zhihu-app

## 简介
zhihu-app 使用了 Node.js + React + Redux + React-router.

##### 此app包括了服务端Express, 为客户端提供api，还包括了客户端，使用React完成，UI主要使用了‘material-ui’


## 开发
```
$ git clone git@github.com:HUJINLIANG/zhihu-app.git
$ cd zhihu-app
$ npm install
$ webpack
$ node server
```
[实例](https://zhihu-app.herokuapp.com/)
